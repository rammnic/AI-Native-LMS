# Промпт: Support L1 → L2 — рост в техподдержке

## Основной промпт для LMS

```
Работаю в техподдержке L1 (Helpdesk), решаю типовые заявки, работаю с Jira/Zendesk. 
Знаю базовые вещи: Windows, простые сетевые проблемы.

Цель: перейти на уровень L2 Support Engineer, уметь диагностировать сложные проблемы и писать скрипты для автоматизации.

Покрыть:
- Продвинутая диагностика Windows (event logs, services, registry)
- Продвинутая диагностика Linux (systemd, logs, processes)
- Глубокие сети (TCP/IP, маршрутизация, firewall, VPN)
- Bash и PowerShell скриптинг
- Основы SQL (запросы, базы данных)
- Мониторинг и alerting (Zabbix, Nagios basics)
- Работа с логами (grep, awk, sed)
- Автоматизация рутины
- Эскалация и взаимодействие с командой
- Документирование решений

Формат: продвинутый курс с реальными кейсами и скриптами для автоматизации.
```

## Ожидаемая структура курса

```
📁 Продвинутая диагностика Windows
   📖 Event Viewer — читать логи как профессионал

   📖 Registry — что трогать можно, что нельзя
   📖 Task Manager и Performance Monitor
   📖 Windows Firewall deep dive
   💻 Practice: Диагностировать bluescreen

   📖 journalctl — логи systemd

   📖 dmesg и syslog
   💻 Practice: Найти причину падения сервиса
```
