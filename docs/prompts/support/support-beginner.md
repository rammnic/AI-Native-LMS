# Промпт: Support Beginner — L1 Helpdesk с нуля

## Основной промпт для LMS

```
Хочу начать карьеру в IT с техподдержки. Никакого опыта в IT нет, умею пользоваться компьютером на уровне обычного пользователя.

Цель: устроиться на позицию L1 Support / Helpdesk и уверенно решать типовые проблемы пользователей.

Покрыть:
- Компьютерные основы для техподдержки (железо, ОС)
- Операционные системы (Windows 10/11, базовый Linux)
- Компьютерные сети базовые (что такое IP, DNS, DHCP, Wi-Fi)
- Как работают тикет-системы (Jira, ServiceNow, Zendesk)
- Типичные проблемы и решения (не подключается Wi-Fi, медленный комп, не открывается сайт)
- Удалённый доступ (AnyDesk, TeamViewer)
- Коммуникация с пользователями — как вежливо объяснять
- Документирование и базы знаний
- Основы информационной безопасности для поддержки
- Эскалация — когда и как передавать заявку

Формат: практический курс с реальными кейсами. Каждая тема заканчивается практическим заданием по решению типовой проблемы.
```

## Ожидаемая структура курса

```
📁 Основы для техподдержки
   📖 Как устроен компьютер (железо)
   📖 Операционные системы (Windows, Linux basics)
   📖 Файловая система и права доступа

📁 Компьютерные сети
   📖 Что такое сеть — локальная и интернет
   📖 IP-адреса и подсети (basics)
   📖 DNS — как работает
   📖 DHCP — автоматическая настройка
   📖 Wi-Fi и проблемы подключения
   💻 Practice: Диагностика сетевых проблем

📁 Работа с тикет-системами
   📖 Что такое тикет и как он живёт

   📖 SLA — время реакции
   💻 Practice: Обработать типичные тикеты

📁 Типовые проблемы и решения
   📖 Компьютер не включается

   📖 Медленная работа системы
   📖 Не открывается сайт

📁 Удалённый доступ
   📖 AnyDesk, TeamViewer, RustDesk

📁 Коммуникация с пользователями

📁 Информационная безопасность

📁 Эскалация

   📖 Советы от опытных инженеров
```
