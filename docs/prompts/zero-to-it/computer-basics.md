# Промпт: Computer Basics — для полных новичков

## Основной промпт для LMS

```
Я полный новичок в IT. Никогда не работал в этой сфере, только играл в игры и пользовался компьютером как обычный пользователь.

Хочу понять основы, чтобы начать путь в IT.

Цель: понять как работает компьютер, интернет, и выбрать направление для обучения (техподдержка, администрирование или разработка).

Покрыть:
- Как работает компьютер (железо — CPU, RAM, SSD)
- Операционные системы (Windows, Linux, macOS)
- Что такое интернет и как работает (простыми словами)
- IP-адреса, DNS, браузер — что происходит когда открываешь сайт
- Файлы и файловая система
- Что такое программирование и код
- Командная строка — не бойся терминала
- Какие профессии бывают в IT
- Как учиться эффективно

Формат: очень простой курс без сложных терминов, с аналогиями из жизни. Каждый концепт объясняется через знакомые вещи. Много практики с реальными действиями на компьютере.
```
