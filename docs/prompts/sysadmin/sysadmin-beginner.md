# Промпт: SysAdmin Beginner — Junior SysAdmin с нуля

## Основной промпт для LMS

```
Хочу стать системным администратором. Опыта в администрировании нет, но знаю базовые вещи: 
пользовался Linux как пользователь, понимаю что такое IP и DNS.

Цель: устроиться на позицию Junior SysAdmin и уверенно управлять серверами.

Покрыть:
- Linux с нуля (установка, настройка, основные дистрибутивы)
- Командная строка и shell (Bash)
- Файловая система Linux
- Пользователи, группы и права доступа
- Управление процессами и сервисами (systemd)
- Сети в Linux (настройка, диагностика)
- SSH и удалённое управление
- Установка и настройка пакетов
- Веб-серверы (Nginx, Apache)
- Базы данных (PostgreSQL, MySQL) — установка и базовая настройка
- Бэкапы — как делать и зачем
- Мониторинг базовый (top, htop, df, du)
- Безопасность сервера (firewall, fail2ban)
- Автоматизация с cron

Формат: пошаговый курс с практикой на виртуальных серверах. Каждая тема закрепляется лабораторной работой.
```

## Ожидаемая структура курса

```
📁 Linux с нуля
   📖 Выбор дистрибутива (Ubuntu, Debian, CentOS, Rocky)
   📖 Установка Linux (VM, облако)
   📖 Первый вход и настройка
   📖 Обновление системы
   💻 Practice: Установить Ubuntu Server

📁 Командная строка
   📖 Терминал — твой главный инструмент
   📖 Базовые команды (ls, cd, pwd, cat, echo)
   📖 Навигация и автодополнение
   📖 Pipes и перенаправление вывода
   📖 Редакторы (nano, vim basics)
   💻 Practice: Навигация без GUI

📁 Файловая система
   📖 Структура директорий (/etc, /var, /home, /tmp)
   📖 Абсолютные и относительные пути
   📖 Создание, копирование, удаление
   📖 Символические ссылки
   💻 Practice: Организовать файлы правильно

📁 Пользователи и права
   📖 Пользователи и группы
   📖 Права rwx (chmod, chown)
   📖 Sudo — временное повышение прав
   📖 SSH keys — авторизация по ключам
   💻 Practice: Настроить доступ для команды

📁 Процессы и сервисы
   📖 Что такое процесс
   📖 ps, top, htop — мониторинг
   📖 systemd — юниты и управление
   📖 systemctl — старт, стоп, статус
   💻 Practice: Управлять сервисами

📁 Сети в Linux
   📖 ip, ifconfig — сетевые интерфейсы
   📖 /etc/hosts, resolv.conf
   📖 traceroute, ping, netstat
   📖 Firewall (ufw basics)
   💻 Practice: Диагностировать сеть

📁 Пакеты и установка софта
   📖 apt/yum/dnf — менеджеры пакетов
   📖 Установка из исходников
   📖 snap, flatpak — контейнерный софт
   💻 Practice: Установить LEMP стек

📁 Веб-сервер Nginx
   📖 Как работает веб-сервер
   📖 Установка и запуск
   📖 Конфигурация виртуальных хостов
   📖 SSL certificates (letsencrypt)
   💻 Practice: Развернуть свой сайт

📁 Базы данных
   📖 PostgreSQL vs MySQL — что выбрать
   📖 Установка и запуск
   📖 Базовые операции (createdb, psql)
   📖 Создание пользователей и баз
   💻 Practice: Настроить БД для приложения

📁 Бэкапы
   📖 Что и когда бэкапить
   📖 rsync — синхронизация файлов
   📖 pg_dump — бэкап PostgreSQL
   📖 Автоматизация бэкапов
   💻 Practice: Настроить ежедневные бэкапы

📁 Безопасность
   📖 Firewall — базовая настройка
   📖 fail2ban — защита от брутфорса
   📖 Обновления безопасности
   📖 Audit logs — кто что делал
   💻 Practice: Защитить сервер

📁 Автоматизация
   📖 cron — планировщик задач
   📖 Написание shell-скриптов
   📖 Systemd таймеры
   💻 Practice: Автоматизировать рутину
```
