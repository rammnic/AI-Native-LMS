# Промпт: DevOps Beginner — Вход в DevOps из IT

## Основной промпт для LMS

```
Хочу перейти из IT Support в DevOps.

Мой опыт: работаю в техподдержке, знаю базовый Linux, понимаю сети, работал с тикет-системами. Знаю что такое Docker и CI/CD на концептуальном уровне (слышал, но не использовал глубоко).

Цель: понять как связана моя работа в поддержке с DevOps практиками и начать двигаться в эту сторону.

Покрыть:
- Как техподдержка связана с DevOps (SRE mindset)
- Углубление в Linux (командная строка, скрипты)
- Основы сетей для DevOps
- Docker — что это и зачем
- CI/CD — что это и как связано с поддержкой
- Мониторинг — основы (что мониторить и зачем)
- Git — базовая работа с репозиториями
- Как начать применять DevOps практики в текущей работе

Формат: практический курс с фокусом на то, как применить знания сразу. Много примеров из жизни техподдержки.
```
