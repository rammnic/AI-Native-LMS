# AI-Native LMS — Чеклист тестирования

**Дата:** 2026-03-21  
**Версия:** 3.0 (объединённый)

---

## Как запустить

```bash
# Запустить все сервисы
docker-compose up -d

# Проверить статус
docker-compose ps
```

**Ожидаемые порты:**
- Frontend: http://localhost:3000
- Backend: http://localhost:8001
- AI Framework: http://localhost:8000
- PostgreSQL: localhost:5432

## ИЗВЕСТНЫЕ ОГРАНИЧЕНИЯ

- AI Framework должен быть доступен для полной проверки (порт 8000)
- При `AI_MOCK_ENABLED=true` контент всегда на английском
- Для real AI нужен `OPENROUTER_API_KEY` в .env
